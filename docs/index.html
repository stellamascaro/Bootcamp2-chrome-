Bootcamp Helper — Extensão MV3

Extensão de exemplo criada como tarefa do Bootcamp II. Nesta página você encontra instruções rápidas, screenshots e um link para download do .zip.

Instalação local (modo desenvolvedor)

  1. Acesse chrome://extensions e ative Developer mode.
  2. Clique em Load unpacked e selecione a pasta do projeto (a raiz onde está o manifest.json).
  3. Abra o popup pelo ícone da extensão e verifique o funcionamento.

Download

Faça o download do arquivo .zip na seção de Releases do repositório.

Permissões

  • storage — para salvar preferências locais.
  • tabs — para possíveis interações com abas (padrão mínimo usado).
